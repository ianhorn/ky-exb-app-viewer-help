Widgets Menu

The main toolbar can contain tools that interact with the map, bring up a widget dialog, or provide information.

Icon Description
The Zoom to Feature widget allows users to zoom to a particular City, County,
or other designated feature such as a State Park or Trailhead.
This widget displays a list of predefined spatial bookmarks. Users can add and
delete bookmarks and those changes are maintained on your computer unitl
you empty your browser cache.
The Base Map widget allows users to select from a gallery of available base
map layers.
The Legend widget shows the symbology of map layers and changes as layers
are turned on or off.
The Map Layers widget shows the layers in the map and allows users to turn
them on, off, change the transparency, zoom to that layer, or even open the attribute table.
The drawing widget allows users to add text or draw on the map. Some versions
also allow you to measure areas and distances as well.
The Add Data widgets enables adding external data to the map
